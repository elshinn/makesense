О курсе

JavaScript — один из самых популярных языков программирования, который используется для создания интерактивных веб-сайтов и приложений. Этот курс разработан специально для начинающих, которые хотят освоить основы программирования и получить практические навыки разработки.

В течение 8 недель вы познакомитесь с основными концепциями JavaScript, научитесь работать с DOM, создавать интерактивные элементы, обрабатывать события и взаимодействовать с API. Каждый урок включает в себя теоретическую часть и практические задания, которые помогут закрепить полученные знания.

К концу курса вы сможете создавать собственные веб-приложения, понимать и модифицировать существующий код, а также получите прочную основу для дальнейшего изучения более сложных тем в JavaScript и фреймворков.

Что вы изучите

  • Основы синтаксиса JavaScript и структуры программ
  • Работу с переменными, типами данных и операторами
  • Функции, объекты и массивы в JavaScript
  • Манипуляции с DOM и обработку событий
  • Работу с API и асинхронное программирование
  • Создание собственных мини-проектов на JavaScript

Требования к курсу

  • Базовые знания HTML и CSS
  • Компьютер с доступом в интернет
  • Любой современный браузер (Chrome, Firefox, Safari)
  • Текстовый редактор (рекомендуется VS Code)
